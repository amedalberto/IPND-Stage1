CSS in General

With CSS you control the styles or presentation of a webpage. HTML should be use only for structure. There are three methods on adding CSS to your page. Method 1 is External CSS, where you add an external sheet with your CSS code to the head section of your html via a link element. Method 2 is Internal CSS, where you add your CSS code to the head element of your web page under the style element. And finally you can add Inline CSS to directly to each element on your web page. However, Inline CSS is not recommended due to code repetition.

Flexbox

Flexbox is a new model of CSS which can be use to position boxes in your webpage. However, in order to support Safari you need to add the prefix -webkit.
